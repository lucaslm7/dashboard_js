📦 Dashboard — E-commerce Olist

Trabalho prático · Disciplina de Desenvolvimento Web · 2º Período

Sobre o projeto: Este dashboard utiliza dados reais do Olist Brazilian E-Commerce Dataset (Kaggle), com 99.441 pedidos entre 2016 e 2018. Os gráficos foram feitos com Chart.js (linha, barra e pizza). As notas são salvas via Back4App (Parse REST API) e o clima vem da Open-Meteo API (pública, sem chave).
